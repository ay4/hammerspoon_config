# Hammerspoon config

## Requirements

- [Hammerspoon](https://www.hammerspoon.org/)
- [Hyperkey](https://hyperkey.app/) — remaps CapsLock to Ctrl+Opt+Cmd

## Hyper key

CapsLock acts as **Hyper** (Ctrl+Opt+Cmd) via the Hyperkey app. All window management uses this modifier.

## Window management

12×8 grid. Screen width is checked at keypress time — screens below 2000px (logical) use the laptop layout; wider screens use the desktop layout. Press **Hyper+/** to see what your screen reports.

| Key | Laptop | Desktop |
|-----|--------|---------|
| Hyper+A | Left half | Center-left column |
| Hyper+D | Right half | Center-right column |
| Hyper+S | Wide center | Small center |
| Hyper+W | Maximize | Wide center |
| Hyper+Q | Messenger top | ← same |
| Hyper+Z | Messenger bottom | ← same |
| Hyper+1 | — | Top bar strip |
| Hyper+2 | — | Bottom bar strip |
| Hyper+E | — | File manager top |
| Hyper+X / C | — | File manager bottom |
| Hyper+K | — | Keynote (full center) |
| Hyper+V | — | Vertical editor |
| Hyper+Enter | Open Warp | ← same |

## App shortcuts

| Key | Action |
|-----|--------|
| F5 | Calendar |
| F6 | Telegram |
| F7 | Telegram Lite |
| Cmd+F6 | Launch & position both Telegram apps |
| F4 | Quit frontmost app |
| Cmd+F4 | Quit all apps with windows |
| Cmd+M | Hide frontmost app |

## Input

| Key | Action |
|-----|--------|
| F13 | Mute |
| F14 | Volume down (hold to repeat) |
| F15 | Volume up (hold to repeat) |
| Horizontal scroll | Switch tabs (Cmd+Shift+[/]) |
| Cmd+Alt+1 | Switch to English layout |
| Cmd+Alt+2 | Switch to Russian layout |
